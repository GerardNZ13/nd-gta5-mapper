    What’s new

    Recent changes in this build:

    • Profiles: separate map data per profile; switch in Map options.
    • POI density heatmap, optional marker clustering, and fade distant territories in Layers.
    • Autosave (~5 min) and Undo import (header) restore state from before the last merge import.
    • Quick + on a territory row prefills POI category; duplicate icon on POI rows clones a pin.
    • Category color palettes in Settings; included in export/import and server sync.
